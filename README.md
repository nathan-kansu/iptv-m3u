# IPTV Playlist

The M3U playlist for IPTV channels.

## Playlist Details

The included channels match the following filters:

- is_nsfw: false
- is_blocked: false
- is_closed: false
- formats: 2060i, 2060p, 1080p, 1080i, 720p
- excluded category: religious

## Files

- playlist.m3u — the main IPTV playlist file

## Usage

Import the M3U file into any IPTV player or app that supports M3U playlists.
